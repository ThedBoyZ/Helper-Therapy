ได้ตามมาตรฐาน 
[ IEEE 802.11 b/g/n และ Bluetooth V4.2 โดยมีโมดูลพื้นฐานคือ ESP-WROOM-32  ซึ่งมี MCU Dual Core แบบ 32 บิต ทำงานที่ความเร็ว 160Mhz SRAM 512K หน่วยความจำ Flash 
สำหรับโปรแกรมขนาด 4M Byte ] 
	- ขาใช้งาน GPIO 36 ขา ความละเอียดในการอ่านค่า ADC 12Bit  สามารถเขียนโปรแกรม ผ่าน Arduino IDE ได้ เชื่อมต่อกับพอร์ต USB ผ่านชิป UART รุ่น CP2102
สำหรับเรียนรู้การเขียนโปรแกรมไมโครคอนโทรลเลอร์เพื่อสื่อสารผ่านอินเตอร์เน็ตหรือบลูธูทด้วย Arduino หรือภาษา LUA  ซึ่งมี library ให้เลือกใช้งานมากมาย
ตัวบอร์ดประกอบด้วย โมดูล ESP-WROOM-32  , ชิปแปลงแรงดัน, ชิปเชื่อมต่อ USB to Serial, Micro USB พร้อมสำหรับการใช้งาน เพียงเสียบสาย USB เชื่อมต่อกับคอมพิวเตอร์ก็พร้อมที่จะเขียนโปรแกรม Arduino 
สั่งการให้ใช้งานได้หลากหลาย เช่นควบคุมอุปกรณ์ไฟฟ้าผ่านอินเตอร์เน็ต, เป็น micro server, เรียนรู้การใช้งานร่วมกับเซนเซอร์ต่างๆ เป็นต้น

Node-Red
1 ) สามารถรับค่าต่าง ๆ เข้ามาพร้อมกันได้อย่างครบถ้วน
2 ) web รองและทำ Appilcation Responsive ได้
3 ) Cloud ฐานข้อมูล / ใช้เป็น mySQL [ mariaDB ] ก็ได้
4 ) มีการเข้าสู่ระบบ login ลงทะเบียน
5 ) รองรับกับ Arduino ได้ดี
6 ) มี mqtt สามารถคุยกันเองภายในตัวอุปกรณ์ได้
7 ) สามารถรองรับ html / json 
8 ) csv รองรับการใช้งานกับโปรแกรมฐานข้อมูลต่างๆ รวมทั้ง Microsoft Excel

9 ) มีการเชื่อมต่อกับ Line Bot
10 ) file in / file out ได้ และยอกมี date picker / text input บลาๆ
